# Resume Entry — myCard Services

## myCard Services — NFC Appointment Booking Platform
**Jan 2026 – Present**
`Next.js 15` `React 19` `TypeScript` `Tailwind CSS v4` `Supabase` `PostgreSQL` `Next.js API Routes`

- Architected and built a full-stack B2B appointment booking platform with **NFC card integration**, enabling clients to tap a card with any smartphone and instantly access a business's services and booking system — no app download required.
- Designed a **three-tier role system** (Superuser, Business Admin, Final Client) with a secure token-based invitation flow, custom session management using cookies and localStorage, and bcrypt password hashing.
- Engineered **30+ RESTful API routes** using Next.js serverless functions backed by Supabase (PostgreSQL), covering appointment scheduling, dynamic availability calculation, service catalog management, and check-in verification.
- Implemented a **dynamic per-business theming engine** that injects custom CSS variables at runtime (colors, fonts, branding) so each business gets a fully branded landing page and booking experience.
- Built a real-time **business admin dashboard** with appointment stats, client management, activity logs, and configurable business hours — all with a mobile-first responsive layout using Tailwind CSS v4.
